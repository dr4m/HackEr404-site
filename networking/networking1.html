    Networking Basics

    Learn the core of computer networking from IP addresses to protocols used on the internet.

    What is Networking?

    Networking refers to the connection of two or more computers or devices to share resources, data, and internet access.


    Key Concepts:

    • IP Address: Unique identifier for a device on a network (e.g., 192.168.1.1).
    • TCP/IP: A suite of protocols that allows computers to communicate.
    • DNS: Converts domain names like hacker404.site into IP addresses.
    • MAC Address: Hardware ID of a network adapter.
    • OSI Model: A conceptual model with 7 layers from Physical to Application.

    Real-World Example

    When you visit hackEr404.site, your browser sends a request to a DNS server to get the IP, then uses TCP/IP to communicate and download the site’s content.


    Protocols You Must Know:

    • HTTP / HTTPS
    • FTP (File Transfer Protocol)
    • SSH (Secure Shell)
    • DNS (Domain Name System)
    • DHCP (Dynamic Host Configuration Protocol)
