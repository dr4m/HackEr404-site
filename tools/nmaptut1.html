    Nmap Tutorial

    Explore and analyze networks with Nmap. Learn scanning techniques and practical use cases for ethical hacking.

    What is Nmap?

    Nmap (Network Mapper) is an open-source tool for network discovery and security auditing. It's widely used in ethical hacking for scanning networks, detecting hosts, open ports, running services, and identifying vulnerabilities.


    Installing Nmap

    • Linux: sudo apt install nmap
    • Windows: Download from nmap.org
    • Termux (Android): pkg install nmap

    Basic Nmap Scan Examples

    • nmap 192.168.1.1 — Scan a single IP
    • nmap 192.168.1.0/24 — Scan a full subnet
    • nmap -sS target.com — Stealth SYN scan
    • nmap -p 1-1000 192.168.1.5 — Scan specific port range
    • nmap -A target.com — Aggressive scan (OS, version, script)

    Top Features of Nmap

    • Port Scanning
    • Service and Version Detection
    • OS Fingerprinting
    • Scriptable Interaction using NSE (Nmap Scripting Engine)
    • Detecting Firewalls and IDS

    Real-World Example

    Let’s say you're testing a website example.com. Use:

    nmap -sV example.com

    This will return open ports and the services running (e.g., Apache, SSH).


    Best Practices

    • Always have legal permission before scanning
    • Use slow timing options (-T2) for stealth
    • Scan during non-peak hours to avoid alerting
    • Combine with other tools like Nikto, Burp, etc.

    Nmap is a must-learn tool for all ethical hackers. Practice scanning in lab environments to master it.
