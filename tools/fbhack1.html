    1. Introduction

    This tutorial explains how phishing works using the Zphisher tool in Termux. It helps you understand how attackers trick users into entering their login credentials. The purpose is to educate and spread awareness about phishing attacks.


    2. What is Zphisher?

    Zphisher is an open-source phishing tool that helps demonstrate how phishing websites are created. It provides templates for popular platforms like Facebook, Instagram, etc.



    3. Download Termux

    To get started, you need to download Termux on your Android device.

    • Termux from Google Play Store
    • Termux from F-Droid (recommended)


    4. Setup Zphisher in Termux

    Open Termux and enter the following commands one by one:

    pkg update && pkg upgrade
    
pkg install git php bash openssh
    
git clone https://github.com/htr-tech/zphisher
    
cd zphisher
    
bash zphisher.sh


    5. Start Phishing (For Testing Purpose Only)

    After launching Zphisher, select the Facebook option from the menu.



    6. Get Phishing Link
