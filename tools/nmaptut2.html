    Nmap Tutorial Part 2 - Advanced Scanning

    Learn advanced scanning techniques with Nmap, including OS detection, stealth scans, and more.

    1. OS Detection

    nmap -O target_ip

    Detect the operating system of the target device.

    2. Aggressive Scan

    nmap -A target_ip

    Perform OS detection, version detection, script scanning, and traceroute in a single command.

    3. Stealth SYN Scan

    nmap -sS target_ip

    Scan with SYN packets to avoid detection by firewalls.

    4. Evasion with Decoys

    nmap -D RND:10 target_ip

    Use decoy packets to mask your real IP and confuse IDS/IPS systems.

    5. Scan Specific Ports

    nmap -p 22,80,443 target_ip

    Scan specific ports on a target device for a more focused analysis.

    6. Timing Templates

    nmap -T4 target_ip

    Speed up the scan with faster timing templates (T0 - T5).
