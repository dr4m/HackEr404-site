Hi! This page helps you learn the most important Kali Linux commands used in ethical hacking, cybersecurity, and penetration testing—perfect for beginners and pros.

Kali Linux Commands List (With Explanations)

1. Basic Navigation Commands

  • pwd - Print current working directory
  • ls - List files in directory
  • cd - Change directory

2. File and Directory Commands

  • mkdir - Make a new directory
  • touch - Create empty file
  • rm - Remove file or directory

3. Package Management (APT)

  • sudo apt update - Update packages list
  • sudo apt install nmap - Install a tool

4. Networking Commands

  • ifconfig - Show network interfaces
  • ping - Test connection to a host

5. Hacking Tools Commands (Kali Specific)

  • nmap - Network scanner
  • sqlmap - SQL injection testing tool
  • msfconsole - Launch Metasploit

6. Useful Shortcuts

  • clear - Clear terminal screen
  • history - Show all used commands
