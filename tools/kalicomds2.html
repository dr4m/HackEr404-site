Advanced Kali Linux Commands

Welcome to the second part of Kali Linux terminal commands. These commands include some powerful tools used in penetration testing.


1. Nmap

Used to scan open ports and detect services on a target.

nmap -sV target.com

2. Metasploit

Powerful framework for developing and executing exploits.

msfconsole

3. Sqlmap

Automates the process of detecting and exploiting SQL injection.

sqlmap -u "http://target.com/page.php?id=1" --dbs


4. Hydra

Brute-force login cracker tool.

hydra -l admin -P /usr/share/wordlists/rockyou.txt ftp://target.com

5. Nikto

Scans websites for vulnerabilities.

nikto -h http://target.com

6. Burp Suite

GUI tool used to intercept and analyze HTTP/S traffic.

burpsuite

7. Netcat

Network tool for debugging and investigation.

nc -lvp 4444

8. John the Ripper

Password cracking tool.

john --wordlist=passwords.txt hashes.txt
