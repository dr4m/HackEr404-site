Wireshark Tutorial Part 1 - Getting Started with Packet Capturing

Learn how to monitor and analyze network traffic with Wireshark step-by-step.

1. What is Wireshark?

Wireshark is one of the most powerful and widely-used network protocol analyzers. It lets you see what's happening on your network at a microscopic level and is an essential tool for ethical hackers, cybersecurity professionals, and network engineers.



2. Installation

Wireshark is available for Windows, macOS, and Linux. You can download it from the official website: www.wireshark.org



3. Capturing Packets

Once installed, open Wireshark and select your active network interface (like Ethernet or Wi-Fi) to begin capturing live traffic.

Click the blue shark fin icon or press Ctrl+E to start capturing.



4. Basic Filters

Wireshark provides powerful filtering options. Here are some useful display filters:

ip.addr == 192.168.1.1

Displays all packets to or from the IP address 192.168.1.1.

http

Filters only HTTP traffic.

tcp.port == 80

Filters all traffic on port 80 (HTTP).



5. Understanding Packet Details

Each captured packet in Wireshark contains three main sections:

  • Packet List Pane – shows summary of packets.
  • Packet Details Pane – expands the selected packet’s protocols and headers.
  • Packet Bytes Pane – shows raw data in hexadecimal.


6. Saving and Exporting

Click on File > Save As to save captured data for future analysis. Wireshark uses the `.pcapng` or `.pcap` format.



7. Bonus: Capture from Terminal using tcpdump

On Linux, you can use `tcpdump` to capture packets directly from the terminal:

sudo tcpdump -i eth0 -w capture.pcap

This command captures traffic on the eth0 interface and saves it in a file called capture.pcap that you can open in Wireshark later.
